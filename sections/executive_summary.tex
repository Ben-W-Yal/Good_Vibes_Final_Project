\section*{Executive Summary}
\addcontentsline{toc}{section}{Executive Summary}

\textbf{Repository URL:} \url{https://github.com/Ben-W-Yal/Good_Vibes_Final_Project}

GeoIntel is an impactful, game-changing intelligence application that redefines how people interact with news and understand modern conflicts. Built with a TypeScript/React and Express stack plus Gemini-powered generative AI, the system fuses social and open-source signals into one operational view so users can move from information overload to actionable understanding. Instead of passively reading disconnected headlines, users engage an interactive geospatial platform that combines event streams, mobility tracking, and AI briefing outputs to interpret what is happening, why it matters, and what is likely to happen next.

The interface pairs narrative intelligence with geographic grounding: grounded web research surfaces relevant article imagery where available, so users see visual context alongside text; users control an interactive 3D globe with terrain and building visualization (via Cesium ion) to pan, tilt, and zoom through the physical environment where events unfold. Selecting a specific event also triggers an \textbf{AI analyst summary} and an optional \textbf{chat} interface for follow-up questions scoped to that event's data. That combination supports a clearer mental map of borders, urban fabric, coastal access, elevation, and regional layout---helping users relate headlines to real geography rather than abstract place names alone.

\textbf{Design rhythm:} the interface keeps a stable geographic and feed context while users incorporate \emph{new} evidence through refreshed markers, regenerated PDB sections, and per-event summaries. That mirrors how analysts work manually---maintain a working picture, then update it when credible signals arrive---without claiming a formal recursive filter or latent-state model in code. The result is still a practical template for faster situational awareness with transparent, AI-assisted synthesis.
